INSTRUCTIONS

Running a program:

Type in your Fortran 90 program into the Editor Pane on the left (The default is a simple Hello World Program). Click the RUN button on the top left of the Output pane to execute the following three-part command in a terminal window, separated by semicolons:

cd ~/Documents/GitHub/Fossil/test;
/usr/local/bin/gfortran -o f [file_name].f90;
./f

The program named file_name is determined with the program name field above the output pane. Change this to match the name given in the editor pane, i.e. PROGRAM [file_name] ... ... END PROGRAM [file_name].

Checking Syntax:

Using the Syntax header, you can check what your Fortran program looks like under the correct syntax coloring. Clicking the CHECK SYNTAX button again refreshes it with changes made to your editor.
